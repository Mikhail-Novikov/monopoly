# Монополия (Monopoly) — демо-проект на React

Это веб-версия настольной игры Монополия, реализованная на React + TypeScript с использованием UI-библиотеки Ant Design. Приложение позволяет добавлять игроков, вести баланс, осуществлять денежные переводы, сохранять игру в локальном хранилище и восстанавливать её после перезагрузки, а также просматривать историю транзакций.

## Технологии

- React 18 + TypeScript
- Ant Design 5
- Webpack 5 (dev и prod сборка)
- localStorage для сохранения состояния игры
- CSS ( Normalize, main.css )

## Установка

1) Убедитесь, что установлен Node.js и npm.
2) Установить зависимости:
   npm install
3) Запуск локального сервера:
   npm run start

## Запуск

Откройте в браузере: http://localhost:2233

## Структура проекта

- src/
  - App.tsx — основной компонент приложения
  - index.tsx — точка входа
  - assets/ — стили и изображения
  - components/ — AddPlayerForm, PlayersList, TransactionsHistory, TransactionForm, StatGame, GameField
  - types/ — типы данных (Player и т.д.)
  - index.html
- README.md
- package.json — зависимости и скрипты

## Особенности

- Поддержка продолжения сохраненной игры через локальное хранилище
- Реализация логики баланса и транзакций между игроками
- Просмотр истории действий (ADD_PLAYER, TRANSACTION, ELIMINATED)
- Возможность сброса состояния игры

## Примечания

- Все данные сохраняются локально в браузере; очистка локального хранилища удалит текущую игру.
